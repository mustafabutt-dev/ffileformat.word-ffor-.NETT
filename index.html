FileFormat.Words for .NET

Welcome to the Homepage of this Open-Source .NET library. FileFormat.Words for .Net is based on OpenXML that is supported by Microsoft. It is very easy & quick to install and offers enterprise-level features to expedite the Word document creation and manipulation processes.

Developers can smoothly equip their business software with a Word documentation automation module by installing this Open-Source .NET API.

Simply run the following command into the NuGet Package Manager and you can start using this .NET library to generate & edit Word documents without using MS Word.

Install-Package FileFormat.Words

It is all about just invoking the methods exposed by FileFormat.Words API. So, install it today to boost productivity and efficiency.

Since, FileFormat.Words library is Open-Source, you can find the source code on the Github repository. In addition, keep an eye on this repo for the latest updates.
